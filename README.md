# SteveGiordano.com

## Tech Stack

### Backend

- NodeJS
- Express.js
- Prisma ORM
- Redis

### Frontend

- Svelte
- esBuild

## How-To

- Make copies of the `.env` and `Caddyfile` from the `examples/` directory in the project root
- `npm install`
- `npm run build:dev`
- `npm run serve:dev`
- `npm run proxy`
